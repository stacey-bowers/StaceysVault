# Gmail Cleanup

## Overview
- **Goal:** Achieve inbox zero on a ~50,000-email Gmail account through a structured, phased cleanup — unsubscribe, delete, archive, organize with labels, then reach zero and maintain it.
- **Why it matters:** Important emails (leads, client replies, critical info) are getting buried. The inbox is a source of mental clutter and a gap in business operations. A clean, organized Gmail is a core part of building tighter systems at Domain Realty Group.
- **Due:** ~May 4, 2026 (medium priority — within one month)
- **Execution model:** Claude does the heavy lifting (surfaces senders, drafts actions, identifies bulk targets). Stacey reviews and approves before anything is touched.

---

## Phases

### Phase 1 — Unsubscribe Sweep
- [ ] Pull top bulk/newsletter/promo senders from Gmail
- [ ] Review sender list and confirm unsubscribe targets
- [ ] Execute unsubscribes (Claude-assisted, Stacey approves)

### Phase 2 — Bulk Delete
- [ ] Identify categories eligible for mass deletion (old promos, expired newsletters, automated notifications, etc.)
- [ ] Review delete targets and confirm scope
- [ ] Execute bulk deletes

### Phase 3 — Archive
- [ ] Define archiving rules (what's worth keeping but out of inbox)
- [ ] Execute bulk archive pass

### Phase 4 — Label & Organize
- [ ] Design new label structure — simple, real estate-focused (start fresh from existing 5 underutilized labels)
- [ ] Review and approve label structure with Stacey
- [ ] Apply labels / set up filters for ongoing routing

### Phase 5 — Inbox Zero + Maintain
- [ ] Process remaining inbox to zero
- [ ] Set up Gmail filters to auto-label and maintain going forward
- [ ] Document maintenance workflow in vault (SOP)
