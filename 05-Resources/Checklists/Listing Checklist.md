# Listing Checklist

## Pre-Listing
- [ ] CMA prepared
- [ ] Listing presentation delivered
- [ ] Listing agreement signed
- [ ] Seller disclosures completed
- [ ] Pre-listing inspection (if applicable)
- [ ] Staging consultation scheduled
- [ ] Repairs / prep work identified

## Preparing to Go Live
- [ ] Professional photos scheduled
- [ ] Virtual tour / video
- [ ] Property description written
- [ ] Flyer / brochure designed
- [ ] Sign and lockbox installed
- [ ] MLS listing entered (hold or active)
- [ ] Showing instructions confirmed

## Active Marketing
- [ ] MLS live — verify all fields
- [ ] Syndication confirmed (Zillow, Realtor.com, etc.)
- [ ] Social media posts published
- [ ] Email blast sent to agent network
- [ ] Open house scheduled & promoted
- [ ] Broker's open (if applicable)

## Under Contract
- [ ] Offer received — present to seller
- [ ] Counter / negotiate
- [ ] Contract executed
- [ ] MLS updated to pending
- [ ] Follow transaction checklist → [[Buyer Transaction Checklist]]

## Post-Closing
- [ ] MLS updated to sold
- [ ] Commission verified
- [ ] Sign and lockbox removed
- [ ] Seller review request sent
- [ ] File archived
