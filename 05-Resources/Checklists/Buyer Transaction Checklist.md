# Buyer Transaction Checklist

## Pre-Contract
- [ ] Buyer consultation completed
- [ ] Buyer agency agreement signed
- [ ] Pre-approval letter obtained
- [ ] Search criteria defined
- [ ] MLS search set up

## Under Contract
- [ ] Contract fully executed
- [ ] Earnest money deposited
- [ ] Title company notified
- [ ] Lender notified with contract
- [ ] Home inspection scheduled
- [ ] Inspection completed — review report
- [ ] Repair request submitted (if applicable)
- [ ] Repair negotiations resolved
- [ ] Appraisal ordered
- [ ] Appraisal received — review value
- [ ] Survey ordered (if applicable)
- [ ] HOA docs reviewed (if applicable)
- [ ] Title commitment received — review
- [ ] Homeowner's insurance bound
- [ ] Loan approval received (clear to close)

## Pre-Closing
- [ ] Final walkthrough scheduled
- [ ] Final walkthrough completed
- [ ] Closing docs reviewed
- [ ] Client reminded of funds to close

## Closing Day
- [ ] Attend closing
- [ ] Keys transferred
- [ ] Congratulations gift / card

## Post-Closing
- [ ] Commission received / verified
- [ ] Client added to post-close drip
- [ ] Review request sent
- [ ] Transaction filed in archive
